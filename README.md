# Sea Turtle Villa Roatan Website

Premium direct-booking website for `seaturtlevillaroatan.com`.

## Stack
- Next.js
- React
- Framer Motion
- Lucide icons
- Static image assets in `/public/images`

## Local setup
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Production deployment recommendation
Deploy to Vercel.

1. Create a free/paid Vercel account.
2. Upload this project to GitHub, then import into Vercel.
3. Set the production domain to `seaturtlevillaroatan.com`.
4. Follow Vercel's DNS instructions at your domain registrar.

## Current integrations
- Book Direct: Ocean One Sea Turtle Villa listing
- Contact: Ocean One contact page
- 3D Tour: Virtuo walkthrough
- Resort: Paradise Beach Hotel
- Reviews: Ocean One Villas Google reviews
